### 🤖 [项目] AuraWell: 多智能体协作的健康管理AI系统
- **项目简介**: 探索多AI Agent协作在健康管理领域的应用，设计并实现了基于 **13个专业AI Agent** 的分布式健康助手系统，解决传统单体AI在复杂健康决策中的局限性。
- **核心功能**: 
  - **多Agent协作架构**: 设计13个专业化AI Agent(推理分析、数据处理、可视化、研究获取等)，实现任务分工与智能协调
  - **深度健康分析**: 集成DeepSeek-R1模型，支持五维度健康评估(营养/运动/体重/睡眠/心理)
  - **实时交互系统**: WebSocket流式对话 + 知识图谱记忆，支持上下文感知的个性化建议
  - **家庭健康管理**: 三级权限体系(Owner/Manager/Viewer)，支持多成员健康数据协作管理
- **技术亮点**: 
  - **分布式AI决策**: 实现多Agent间的任务分解、并行处理与结果融合
  - **个人健康画像**: 基于知识图谱构建用户健康档案，支持历史数据追踪
  - **混合存储架构**: SQLite本地存储 + 云端AI推理，平衡隐私保护与计算能力
- **技术挑战**: 解决了多Agent协调一致性、健康数据建模、实时流式推理、复杂权限控制等关键技术问题
- **技术栈**: `Python`, `Vue 3 + TypeScript`, `DeepSeek-R1`, `Multi-Agent System`, `FastAPI`, `WebSocket`, `JWT`, `SQLite`, `阿里云DashScope`

- **技术实现**: 
    - **模型设计**: 结合Transformer编码器和MAML梯度更新机制，构建具备快速适应能力的智能体
    - **实验验证**: 在多个不同域的资源分配任务上验证模型的泛化能力和适应速度
    - **算法优化**: 针对收敛稳定性和计算效率进行算法改进
- **技术栈**: `Python`, `PyTorch`, `Meta-Learning (MAML)`, `Deep Reinforcement Learning (DQN)`, `Transformer Architecture`
